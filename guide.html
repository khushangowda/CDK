Voice Command Guide

  • "Scroll down" → Scrolls the page down
  • "Scroll up" → Scrolls the page up
  • "Go back" → Navigates to the previous page
  • "Go forward" → Moves to the next page
  • "Open new tab" → Opens a new browser tab
  • "Search for [query]" → Searches the page
  • "Search Google for [query]" → Opens a Google search
  • "Click [button name]" → Clicks a button
  • "Zoom in" → Increases zoom level
  • "Zoom out" → Decreases zoom level
  • "Refresh" → Reloads the page
  • "Close tab" → Closes the current tab
  • "Guide" → Opens this guide
  • "Back" or "Close" → Closes this guide
